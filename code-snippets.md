## HTML Page Shell

Copy this html code and paste it into an empty document named `index.html` to create a webpage with nothing in it except a title.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <link rel="stylesheet" href="styles.css">
  <title>Page Title</title>
</head>
<body>
  <h1>Title</h1>
</body>
</html>

```
